What Halyard knows

Solid green means the radio reported it. Dashed amber means Halyard put it on the wire and got no acknowledgement. Dotted grey means it has not been sent at all.

This bandscope is built by actually retuning the radio, one step at a time, and reading the S-meter at each stop. It is slow, it takes the receiver away from you while it runs, and the S-meter only has 16 levels, so the trace is a coarse staircase and not a spectrum analyser. Open the squelch first or every stop reads zero.

Halyard ships with the FT-897D protocol as it is described in print. Several of those descriptions disagree with each other, and none of them are your radio. These trials replace each assumption with a measurement. Work through them once with the radio on the bench, and the constants below stop being guesses.

Virtual radio

A software FT-897D that speaks the same 5 byte protocol, so Halyard can be exercised without a cable. Fault injection makes it misbehave the way real serial links do.

Status byte polarity

Published descriptions of the FT-897D status bytes disagree on which way round several bits read. These are the manual overrides. To settle a polarity by measurement instead of by guesswork, run the matching trial on the Trials station, which sets these for you and records why.
